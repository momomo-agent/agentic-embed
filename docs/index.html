Basic usage
import { create } from 'agentic-embed' const store = create({ apiKey: process.env.OPENAI_API_KEY }) await store.add('doc-1', 'Quantum computing uses qubits...') await store.add('doc-2', 'Neural networks mimic the brain...') const results = await store.search('How do qubits work?') // → [{ id: 'doc-1', chunk: '...', score: 0.89 }]
RAG with agentic-lite
import { ask } from 'agentic-lite' import { create } from 'agentic-embed' const store = create({ apiKey: '...' }) await store.add('docs', longDocument) const ctx = await store.search(question, { topK: 3 }) const result = await ask( `Context:\n${ctx.map(r => r.chunk).join('\n')}\n\n${question}`, { apiKey: '...' } )

What's inside

✂️ Auto-chunking

Smart text splitting on paragraph and sentence boundaries. Configurable size and overlap.

🔌 OpenAI + any proxy

text-embedding-3-small by default. Custom baseUrl for Voyager, Cohere, or self-hosted.

🏠 Local mode

TF-IDF vectors — no API key, no network. Good enough for testing and small datasets.

📊 Similarity scoring

Cosine similarity with threshold filtering, metadata filtering, and per-document deduplication.

📦 Export / Import

Serialize the full vector store. Cache embeddings, move between environments.

⚡ Batch embedding

addMany() sends chunks in configurable batches. Handles rate limits with retry.
